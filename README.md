# React mui blog app

## Clone the repo
```
  git clone https://github.com/Shoebtamboli/React-mui-blog-app.git
```

## Navigate to directory
```
  cd React-mui-blog-app
```

## Install dependencies and run the app

```
  npm install
  npm start
```
